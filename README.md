## Getting Started

### Via Cloning The Repository:

```
# Get the app locally
git clone https://github.com/kingisaac95/react-redux-starter-kit.git

# Change directory
cd react-redux-starter-kit

# Install Package dependencies
npm install

# Run the application
npm start

# Run test
npm run test
```
